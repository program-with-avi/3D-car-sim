# 3D Car-sim 🚗  
This is a 3D car simulator where you can drift or just roam around the world.    
This game was entirely developed using [Python](https://www.python.org/) language.  
## Instalation⬇️  
If you already have python installed go to Python library installation (ursina)  
You could also directly download the ```.exe``` file and run it directly(only for windows users)  
### Installing python  
To install [Python](https://www.python.org) go to python's official website for download [https://www.python.org/downloads/](https://www.python.org/downloads/) and install it according to your OS. And make sure you installed version 3.11 or above.  
### Python library installation (ursina)  
After completing the [installation of python](https://www.python.org/downloads/), open your command prompt (cmd or powershell in Windows, terminal in Mac/Linux) and type ```pip install ursina```.  
## Running the game🎮  
After you have completed the [installation of Python](https://www.python.org/downloads/) and have library installed, now you can run the game.  
1. Duplicate this repository or just download the latest version folder  
2. Now open the ```.py``` file and run it.  
3. Congrats you have successfully launched the game!  
4. You could also directly download the ```.exe``` file and run it directly(only for windows users)  
### Using the game  
- Use keys ```W, S, D, A ``` to control the movement of the car.
